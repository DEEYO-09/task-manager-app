# Task Manager System

A Flask-based role-based task management web application.

## Live Demo

https://task-manager-app-2-3d6c.onrender.com

## Features

- HR & Employee Login
- Task Assignment
- Productivity Tracking
- Deadline Monitoring
- Priority Classification
- Dashboard Analytics

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- Bootstrap

## Run Locally

```bash
pip install -r requirements.txt
python app.py
