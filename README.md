# TASK MANAGER WEB APPLICATION

## Description:
This is a full-stack Task Manager web application built using Flask, SQLite, HTML, CSS, and Bootstrap. It supports role-based access for HR and Employees to manage and track tasks efficiently.

---

## Features:
- User Signup and Login system (HR & Employee roles)
- HR dashboard to assign and manage tasks
- Employee dashboard to view and update tasks
- Update task status (Pending / Done)
- Automatic overdue task detection
- Task filtering based on status (All / Pending / Done / Overdue)
- Dashboard with task statistics and productivity insights
- Session-based user authentication

---

## Tech Stack:
- Python (Flask)
- SQLite Database
- SQLAlchemy ORM
- HTML, CSS, Bootstrap 5

---

## How to Run:
1. Install dependencies:
   pip install flask flask_sqlalchemy

2. Run the application:
   python app.py

3. Open in browser:
   http://127.0.0.1:10000

---

## Project Overview:
This project demonstrates full-stack web development with role-based access control, database integration, and dynamic task management using Flask.
